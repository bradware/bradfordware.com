VotingViz | Spring 2016

VotingViz is a data visualization dashboard to present the 2016 Presidential Primary results of each state. An easy to use interface where you simply click on states in the U.S. map to see the current standings.

Used D3.js, jQuery, Bootstrap, Bower, Topojson, and responsive design principles to build the web application. VotingViz on Github

Twitter Data Pipelines | Spring 2016

Built reactive data pipelining application using Scala to stream live tweets about MailChimp. Each phase of the pipeline transforms the data, where the JSON tweet is serialized using Thrift and eventually transformed into a simpler data objects for storage.

Used technologies such as Reactive Kafka, Twitter API, Thrift, and Akka Streams. Twitter Data Pipelines on Github

Scrubs | Fall 2015

Scrubs is your mobile health companion while traveling abroad. A mobile app designed to handle all of the health requirements when traveling to different nations, but also to store your personal medical history. Went through the complete design phase of the application including user research, mockup alternatives, a prototype, and a testing plan.

Prototype design by Pranil Vora. Scrubs on Github
